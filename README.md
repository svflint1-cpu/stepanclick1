# Stepan Clicker — optimized build

## Что внутри
- `index.html` — лёгкий HTML без огромных base64-картинок.
- `assets/` — картинки, иконки, музыка и фоновые изображения.

## Как залить на GitHub Pages
1. Создай репозиторий.
2. Загрузи `index.html` и папку `assets`.
3. Открой Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: main / root.
6. Сохрани и дождись ссылки.

## Важно
Не загружай только `index.html`. Папка `assets` обязательна.
