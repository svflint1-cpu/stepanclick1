# Stepan Clicker

Готовая Telegram WebApp / HTML-игра.

## Структура

- `index.html` — игра
- `assets/` — картинки и музыка, вынесенные из base64

## Как залить на GitHub Pages

1. Создай репозиторий на GitHub.
2. Загрузи все файлы из архива: `index.html` и папку `assets`.
3. Открой: Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: main / root.
6. Сохрани.
7. Через минуту GitHub даст ссылку вида:
   `https://USERNAME.github.io/REPO/`

## Важно

Не загружай только `index.html`. Папка `assets` должна лежать рядом.
